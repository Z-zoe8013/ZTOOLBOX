# ztoolbox
-安卓版本支持从下拉快捷方式中快速复制云端数据到剪切板，无需启动应用-注意部分手机需要授权剪切板后台写入权限
-已在小米手机完美使用

-数据安全-上传到云的数据经过加密再上传，密钥仅在本地存储，建议设置密钥长度大于16位
